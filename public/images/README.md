# Imágenes del Portfolio TidyTouch

## Estructura de Imágenes según Brief

Coloca las imágenes recortadas del PDF en esta estructura exacta:

```
public/images/
├── hero-perchero.jpg          # Slide 1: Perchero de ropa con espejo en el suelo (Hero Section)
├── toallas-beige.jpg          # Slide 2: Toallas beige sobre superficie blanca (Franja de Propuesta de Valor)
├── cajas-lavanda.jpg          # Slide 3: Foto cenital de cajas blancas y rama de lavanda (Misión)
├── perchero-industrial.jpg    # Slide 4: Perchero industrial negro con ropa colgada (Filosofía)
├── luisa-roca.jpg             # Slide 5: Luisa sentada en las rocas (Bio)
├── collage-hogar.jpg          # Slide 6: Collage (Despensa, Cajón, Archivos, Toallas) - Organización de Hogar
├── archivadores-blancos.jpg   # Slide 7: Archivadores blancos (Oficina)
├── manos-laptop.jpg           # Slide 7: Manos en laptop (Digital)
├── contacto-vertical.jpg      # Slide Contact: Columna vertical (Armario, Planta, Armario ordenado)
└── banner-inspiracion.jpg     # Slide 9: Banner horizontal (Toallas, Perchas, Despensa, Lavandería)
```

## Notas Importantes

1. **NO usar stock photos genéricas** - Solo las imágenes del PDF
2. **Resolución**: Las imágenes pueden no ser 4K. Se usarán con `max-width` en CSS para evitar pixelado
3. **Marcos blancos**: Se añadirán alrededor de las imágenes para disimular baja resolución si es necesario

## Orden de Implementación

1. Hero Section → `hero-perchero.jpg`
2. Value Proposition → `toallas-beige.jpg`
3. Sobre Mí - Misión → `cajas-lavanda.jpg`
4. Sobre Mí - Filosofía → `perchero-industrial.jpg`
5. Sobre Mí - Bio → `luisa-roca.jpg`
6. Servicios - Hogar → `collage-hogar.jpg`
7. Servicios - Oficina/Digital → `archivadores-blancos.jpg`, `manos-laptop.jpg`
8. Contacto → `contacto-vertical.jpg`
9. Inspiración (Footer/Optional) → `banner-inspiracion.jpg`
